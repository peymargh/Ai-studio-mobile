# Contributing to AI Studio Mobile

Thank you for considering contributing to AI Studio Mobile! This document outlines the guidelines for contributing to this project.

## Forking the Repository
1. Go to the [AI Studio Mobile GitHub page](https://github.com/peymairgh/ai-studio-mobile).
2. Click the **Fork** button at the top-right corner of the page.
3. This will create a copy of the repository in your own GitHub account.

## Creating a Feature Branch
1. Clone your forked repository to your local machine:
   ```
   git clone https://github.com/YOUR_USERNAME/ai-studio-mobile.git
   ```
2. Navigate into the project directory:
   ```
   cd ai-studio-mobile
   ```
3. Create a new feature branch:
   ```
   git checkout -b feature/your-feature-name
   ```

## Making Changes
- Make your changes in the code base.
- Be sure to write clear, concise commit messages that explain your changes.

## Submitting a Pull Request
1. Push your changes to your forked repository:
   ```
   git push origin feature/your-feature-name
   ```
2. Go to the **Pull Requests** tab on the original repository
3. Click the **New Pull Request** button.
4. Choose the base branch (usually `main`) and your feature branch, and review the differences.
5. Write a description of the changes and click **Create Pull Request**.

Thank you for contributing! We appreciate your help in improving AI Studio Mobile!
