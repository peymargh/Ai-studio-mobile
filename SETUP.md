# Setup Instructions

## Installation Guide

1. **Clone the repository:**  
   Run the following command in your terminal:
   ```bash
   git clone https://github.com/peyamargh/ai-studio-mobile.git
   ```

2. **Navigate to the project directory:**  
   ```bash
   cd ai-studio-mobile
   ```

3. **Install dependencies:**  
   For Node.js projects:
   ```bash
   npm install
   ```
   or if you are using Yarn:
   ```bash
   yarn install
   ```

## Setup Guide

1. **Environment Variables:**  
   Make sure to configure your environment variables as per the `.env.example` file.

2. **Run the application:**  
   Use the following command to start the application:
   ```bash
   npm start
   ```
3. **Build the application:**  
   To create a production build, run:
   ```bash
   npm run build
   ```

4. **Testing:**  
   Ensure to run the test suite:
   ```bash
   npm test
   ```
